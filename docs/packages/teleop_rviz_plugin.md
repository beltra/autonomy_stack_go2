# teleop_rviz_plugin

## Ruolo
Pannello RViz base per smart-joystick e comando “Resume Navigation to Goal”.

## Quando viene usato
Stack base o route quando si vuole controllo GUI minimale in RViz.

## Topic
- Publish: `/joy`

## Dati trasmessi
- assi joystick sintetici (velocità lineare/yaw)
- pulsante modalità per riprendere navigazione verso goal/waypoint

## Parametri
Nessuno (plugin UI).
